# 仿-Vue开发去哪儿网手机页面



### 项目启动

``` 
# 拷贝项目到本地
git clone git@github.com:LuoYupeng/travel.git

# 安装依赖
npm install

# 在localhost:8080启动本地服务器
npm run dev

```
>   Vue单页面开发应用

>   由axios负责数据请求，Vue-router负责路由转发，Vuex实现状态管理
  

### 应用技术
- Vue、Vue-router、Vuex
- 插件axios、vue-awesome-swiper、better-scroll
- stylus:CSS预处理器
